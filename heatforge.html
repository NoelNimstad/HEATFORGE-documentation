§ Description

The Heatforge class functions as the main class, internal rendering is routed through the Heatforge class, and some external rendering too. The Heatforge class is needed for your game to function and is core to the Heatforge game engine.

¶ void Start()

Starts the engine's internal frame loop, which gathers input, calls hooks and calculates delta time. Will run until bool running is false. Generally called inside the int main() function. See code example at ¶ void End().

¶ void End()

Runs cleanup, generally after bool running is set to false. Usually put in the int main() function after the void Start() function.

Heatforge::Heatforge _("demo", 320, 240, 3, 16);
int main()
{
    _.Start();
    _.End();
}

¶ void Clear()

Clears the screen with the int clearColor[3] variable, which is by default black. See code example at ¶ void Update().

¶ void Update()

Pushes render instructions made to the screen, showing the "present".

Heatforge::Heatforge _("demo", 320, 240, 3, 16);
void Draw() // assuming Draw() is a hooked function
{
    _.Clear();
    _.Update();
}

¶ void DoVSync(bool state)

Enables or disables VSync.

¶ void Hook(std::function<void()> hook)

Hooks a function into the engine's main loop, calling each hook once per frame.

// Simple program that will write "frame" to the console once every 60th of a second.

Heatforge::Heatforge _("demo", 320, 240, 3, 16);

void Frame() 
{
    std::cout << "frame" << std::endl;
}

int main()
{
    _.Hook(&Frame);
    _.Start();
    _.End();
}

¶ Heatforge::Spritesheet LoadSpritesheet(char const * path, int tileWidth, int tileHeight)

Loads a png image from the path char const * path as a spritesheet and sets up tiling every int tileWidth and int tileHeight. If you are loading a single image, set int tileWidth and int tileHeight to the image's width and height respectively.

Heatforge::Heatforge _("demo", 320, 240, 3, 16);
Heatforge::Spritesheet player = _.LoadSpritesheet("path/to/player.png", 20, 20);

¶ bool running

By default set to true, if bool running is ever set to false, the Start function will terminate giving way to cleanup and/or exit code, like void End().

¶ int clearColor[3]

By default set to { 0, 0, 0 } (black). This is the color which the void Clear() function uses to clear the screen with, in other words, the background color of the game.

¶ Uint32 delta

The game's delta time in ms (between frames). Calculated within the "main" function started by void Start().

¶ std::unordered_map<int, bool> inputs

Keyboard inputs as a bool stored by SDL's keyboard ids. See code example at ¶ std::unordered_map<const wchar_t *, int> keys.

¶ std::unordered_map<const wchar_t *, int> keys

A map mapping a string, in this case a key name, to their respective key code which can be used as a lookup in the std::unordered_map<int, bool> inputs.

Heatforge::Heatforge _("demo", 320, 240, 3, 16);

void Frame()
{
    if(_.inputs[_.keys[L"w"]])
    {
        std::cout << "banana" << std::endl;
    }
}
