class Heatforge::Font

§ Description

Class that loads HFont (HeatFont) files into a game such as to allow the ability to render text to the screen. MiniStad.hfont is available for download at [coming soon].

¶ void Draw(std::vector<int> * integerArray, Heatforge::Vector2 * position)

Draws text from an std::vector<int> containing character indexes and a class Heatforge::Vector2 providing the base position. See code example at ¶ std::vector<int> ConvertStringToIntArray(std::wstring string).

¶ std::vector<int> ConvertStringToIntArray(std::wstring string)

Converts a std::wstring to a std::vector<int> from information loaded from the HeatFont file.

Heatforge::Heatforge _("demo", 320, 240, 3, 16);
Heatforge::Font font(L"path/to/font.hfont", &_);

Heatforge::Vector2 position(100, 100);
std::vector<int> text = font.ConvertStringToIntArray(L"banana");

void Frame()
{
    _.Clear();

    font.Draw(&text, &position);

    _.Update();
}
