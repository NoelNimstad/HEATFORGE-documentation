§ Welcome to the Heatforge documentation!

Here you will find all official information related to development with the Heatforge game engine. The Heatforge engine is still very early on in development and will not be finished for a long time, so expect to run into issues and bug on the way; and if you do run into bugs and issues, or extend the engine to your own liking, please feel free to make a pull request on the github repository.

To the left you will find all classes implemented in the engine, both internal and external and to the right you will find guides related to the engine.
